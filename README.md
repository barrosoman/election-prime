## :fountain_pen: About
Election Prime is a political survey research analysis website with the goal to assist the candidate's political team providing solutions and strategies aiming the candidate victory. In summary, the website receive a survey research made by the candidate team and provides comparative infographics, statistics measures and insights about that data sample.

## 🎓 A college project
The Election Prime project was made as a requirement for the **Probability and Statistics** course.

## 🚀 Techs
The techs used to create the website are

* Vue 3
* TypeScript
* ApexCharts

## ⬇️ Installing
#### Installing all necessary packages to run the website
```
npm install
```

#### Run the website locally for development with hot-reloading.
```
npm run dev
```

#### Compiles and minifies for production
```
npm run build
```

#### Fixed files and lints
```
npm run lint
```
